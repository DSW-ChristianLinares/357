Proposito del sitio



El proposito de la pagina es impulsar y reforzar al aprendizaje de la tematica "Aprendiendo etiquetas básicas de HTML, atributos y valores", para lograr comprender la tematica de manera que podamos avanzar a los siguientes temas para el desarrollo de paginas web.



Etiquetas



Las etiquetas HTML son fragmentos de código que permiten crear elementos HTML, estructuras básicas del lenguaje de programación HTML en el que se escriben las páginas web porque es el que entienden los navegadores.

Además de las etiquetas, los elementos HTML tienen dos propiedades básicas:

1. Atributos, que se encuentran en la etiqueta de inicio.
2. Contenido, ubicado entre las dos etiquetas.
